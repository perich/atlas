# Atlas SPA Boilerplate

A lean React SPA starter with pinned dependencies, Vite, TanStack Router, Tailwind CSS, Vitest, Playwright, oxlint, and oxfmt.

## Scripts

```txt
pnpm dev
pnpm build
pnpm typecheck
pnpm lint
pnpm format
pnpm test:run
pnpm test:e2e
```

## Stack

- React + TypeScript
- Vite
- TanStack Router
- Tailwind CSS
- Radix-ready component primitives
- Vitest + Playwright
- oxlint + oxfmt
